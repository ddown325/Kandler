## Deploy this repo to GitHub Pages

1. Push this repo to https://github.com/ddown325/Kandler
2. Go to **Settings → Pages → Source → select "GitHub Actions"** (NOT "Deploy from a branch")
3. Wait ~2 minutes — your site goes live at https://ddown325.github.io/Kandler/

See **QUICK-START.md** for step-by-step instructions.
