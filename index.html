Open Kandler View on GitHub

Don't see the app? GitHub Pages might be misconfigured.
  1. Go to your repo's Settings → Pages
  2. Under Source, select GitHub Actions (NOT "Deploy from a branch")
  3. Check the Actions tab — the "Deploy to GitHub Pages" workflow should be green ✓
  4. If it's red ✗, click into it to see the build error
